# Cat-as-a-Service: Yandex.Disk Uploader

Курсовая работа по API-интеграции. Скрипт генерирует изображение котика с произвольным текстом и загружает его на Яндекс.Диск.

## 🚀 Возможности

- Генерация изображения котика с текстом через [cataas.com](https://cataas.com)
- Загрузка на Яндекс.Диск через REST API
- Прогресс-бар загрузки (`tqdm`)
- Сохранение истории загрузок в `image_info.json`
- Автоматическая очистка имени файла от спецсимволов

## 🛠 Стек технологий

- Python 3
- requests (работа с API)
- tqdm (прогресс-бар)
- Yandex.Disk API
- Cat-as-a-Service API

## 📦 Установка и запуск

1. **Клонировать репозиторий**

```bash
git clone https://github.com/Stendal16471/Coursework-API.git
```

```bash
cd Coursework-API
```

2. **Создать виртуальное окружение**
   
```bash
python -m venv venv
```

```bash
source venv/bin/activate  # для Linux/Mac
```

```bash
venv\Scripts\activate     # для Windows
```

3. **Установить зависимости**

```bash
pip install -r requirements.txt
```

4. **Получить токен Яндекс.Диска**

Перейдите на Полигон Яндекс.Диска
Скопируйте ваш OAuth-токен

5. **Запустить скрипт**

```bash
python code.py
```

6. **Ввести данные**

Введите любой текст (например, "Привет, мир!")
Вставьте токен Яндекс.Диска

Токен не хранится в коде, а вводится при каждом запуске

Папка на Яндекс.Диске создаётся автоматически (SPD-132)
История загрузок сохраняется локально
